## Common Pitfalls

### Pitfall 1: Missing test directory
**What goes wrong:** Phase 2 tests can't be created
**Why it happens:** test/ directory was deleted or not created
**How to avoid:** Ensure test/ directory exists before Phase 2
**Warning signs:** Missing test/ folder in project root

### Pitfall 2: Incorrect Vite entry point
**What goes wrong:** Build fails or produces wrong output
**Why it happens:** Entry point path doesn't match source location
**How to avoid:** Verify `lib.entry` in vite.config.mjs points to `src/lib/heatmap.js`
**Warning signs:** Build output doesn't contain expected files

### Pitfall 3: CSS not emitted to dist/
**What goes wrong:** Styles not included in built library
**Why it happens:** CSS import missing in source or build config incorrect
**How to avoid:** Import CSS in heatmap.js: `import './heatmap.css'` or use vite-plugin-lib-inject-css
**Verification:** Check dist/d3-heatmap2.css exists after build
